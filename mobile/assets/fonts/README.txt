Download Poppins font files from Google Fonts (https://fonts.google.com/specimen/Poppins) and place the following files in this folder:

- Poppins-Regular.ttf
- Poppins-Bold.ttf
- Poppins-SemiBold.ttf
- Poppins-Medium.ttf
- Poppins-Light.ttf

This is required for the app to use the Poppins font as specified in pubspec.yaml.
